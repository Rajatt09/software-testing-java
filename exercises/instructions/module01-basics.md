# Module 01 Exercise Instructions

## Task
Add one more unit test for `Order.totalAmount()` and one error-path test for invalid `OrderItem` input.

## Constraints
Use only JUnit assertions and keep the test focused on one behavior at a time.

The new tests should describe exactly what total or exception is expected.

## Hints
